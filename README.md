## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/felipealvescosta/skeleton-mobile
```

2. Install packages

```sh
yarn install
```

3. Run Project

```sh
react-native run-ios or react-native run-android
```
